Методы Telegram SDK

Список методов извлечения данных

  • window.Telegram.WebApp.ready() - сообщает, что приложение готово.
  • window.Telegram.WebApp.initData - строка с данными инициализации (для сервера).
  • window.Telegram.WebApp.initDataUnsafe - объект с данными инициализации.
  • window.Telegram.WebApp.initDataUnsafe.user - объект пользователя Telegram.
  • window.Telegram.WebApp.themeParams - параметры текущей темы оформления.
  • window.Telegram.WebApp.colorScheme - цветовая схема (light/dark).
  • window.Telegram.WebApp.version - версия Telegram Bot API.
  • window.Telegram.WebApp.platform - платформа запуска (android, ios, desktop, etc).
  • window.Telegram.WebApp.viewportHeight - высота видимой области приложения.
  • window.Telegram.WebApp.viewportVisibleHeight - высота видимой области, за вычетом перекрытых элементов.
  • window.Telegram.WebApp.isExpanded - развернуто ли приложение на весь экран.
  • window.Telegram.WebApp.backgroundColor - основной цвет фона.
  • window.Telegram.WebApp.headerColor - цвет заголовка.
