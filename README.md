# trader
Экспериментальный трейдинг-бот.  Демонстрация связки Telegram Mini App → локальный n8n → JS / Lua → QUIK → брокеры → биржи. Арбитраж, автоматизация и немного инженерной магии.

### n8n: Тестирование
Страница проекта: [index.html](./index.html) (через GitHub Pages).
Страница тестов Mixed Content: [tests.html](./tests.html).

В `tests.html` зафиксированы результаты тестирования различных способов обхода Mixed Content для связи с локальным n8n.
